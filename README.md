# Personal Portfolio Website

This repository contains a basic personal portfolio website built using HTML, CSS, and JavaScript. The site highlights personal information, skills, projects, and contact details in a clean and responsive design.

## Features
- Simple and modern interface
- Fully responsive design for all devices
- About, Skills, Projects, and Contact sections
- Easily customizable structure and style
- Lightweight and fast loading

## Technologies Used
- HTML5 for structure  
- CSS3 for styling and layout  
- JavaScript for interactivity  

## Installation
1. Clone or download the repository.  
2. Open the folder in your code editor.  
3. Run `index.html` in your browser.  

## Customization
- Update personal details and images in `index.html`  
- Modify styles in `style.css` to match your theme  
- Add or remove project cards in the Projects section  

## Project Goals
This project serves as a beginner-friendly template for creating personal portfolio websites. It is ideal for showcasing professional work, personal achievements, or web development learning progress.
